# Project Roadmap & Implementation Plan

## 1. Phase 1: The Core "Workbench" (MVP)
**Goal:** Create a stable, collaborative canvas where users can generate and move stickies in real-time.
*   **Milestone 1.1**: Repo setup, CI/CD pipelines, and boilerplate architecture.
*   **Milestone 1.2**: Authentication, User Sessions, and Dashboard.
*   **Milestone 1.3**: Real-time canvas rendering (CRDTs integration, panning/zooming).
*   **Milestone 1.4**: Basic shape placement (sticky notes, text bounding boxes).

## 2. Phase 2: The "AI Facilitator" Integration
**Goal:** Introduce the language model to auto-cluster and analyze the canvas data.
*   **Milestone 2.1**: Vectorization pipeline (turning stickies into embeddings).
*   **Milestone 2.2**: Right-panel AI chat interface contextually aware of board state.
*   **Milestone 2.3**: Automated clustering logic and "Magic" UX improvements.

## 3. Phase 3: External Connectivity (Miro & Notion)
**Goal:** Prove the value of a "Unified" workbench by sending the data outwards.
*   **Milestone 3.1**: Notion OAuth integration and report generation logic.
*   **Milestone 3.2**: Miro bidirectional (or one-way) sync for existing assets.

## 4. Phase 4: Mobile & Accessibility 
**Goal:** Allow remote participants to engage via mobile devices and harden the app.
*   **Milestone 4.1**: React Native / Expo auxiliary application for rapid ideation "throwing" ideas.
*   **Milestone 4.2**: Security audits and performance hardening.
